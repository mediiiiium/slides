## 資料を追加するには

1. `your-deck.html`（自己完結の HTML スライド）をこのリポジトリ直下に置く
2. `index.html` の `<!-- ==== DECK CARD ==== -->` ブロックをコピーして `href` / `h2` / `date` / `p` を編集
3. commit & push（Pages に自動反映）

各スライドはタップ／スワイプ／矢印キーでめくれる単一 HTML ファイル。外部依存は Google Fonts のみ。
